## Bitcoin Padawans - Seed Backup Template

Este repositório contém um template em HTML para **backup de seed de carteira Bitcoin**, feito para ser **impresso em papel**.

O template faz parte do projeto **Bitcoin Padawans**, uma iniciativa de **educação em Bitcoin** de mesas gratuitas ensinando à configurar carteiras Bitcoin. Vou começar montando essas mesas na minha própria cidade e compartilhando os resultados online, para que mais pessoas possam aprender sobre Bitcoin.

A página `index.html` é um **modelo de cartão de backup de seed**. Ela existe para que **qualquer pessoa possa acessar o arquivo, imprimir seus próprios cartões e usá-los no aprendizado e autocustódia**, rumo a se tornar um verdadeiro **Mestre Jedi Bitcoin**, e não apenas um padawan.

Sinta-se à vontade para copiar, adaptar e imprimir este template nas suas próprias iniciativas de educação em Bitcoin.
