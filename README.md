## Bitcoin Padawans – Site do Projeto

Este repositório contém o **site principal do projeto Bitcoin Padawans**, iniciativa de educação em Bitcoin focada em **explicar Bitcoin para leigos**, organizar **mesas abertas**, eventos e materiais didáticos prontos para impressão.

Atualmente o site oferece:

- **Página inicial (`index.html`)**: apresentação do projeto, visão geral e links para materiais.
- **Gerador de panfletos (`/panfleto/`)**: página para montar e imprimir panfletos de eventos/mesas abertas, com opções de layout, recorrência e cores.
- **Materiais de apoio**: links para tutoriais de carteiras, material sobre Bitcoin e Lightning, estatísticas de rede e backup de seed.
- **Ferramentas de backup de seed (`/seed/backup/`)**: modelos em HTML pensados para impressão, ajudando na autocustódia.

O objetivo é que qualquer pessoa possa:

- Reutilizar o site como base para **atividades locais de educação em Bitcoin**.
- Imprimir panfletos e cartões de backup de seed de forma simples.
- Adaptar o conteúdo para a realidade da sua cidade/comunidade.
